# Skill Evolution Guidelines

These skills are a good starting point, but they may not fully meet the needs of users from different domains.

## Expectations

- Adapt and evolve existing skills to better fit the user's context and goals.
- Use `/skill-creator` when you need to create a new skill or improve an existing one.

## Rules

- Every new skill must be added to the `.claude/skills/` directory.
- Every time you evolve a skill, update its `SKILL.md` with the new version.
- When updating a skill, clearly explain why the skill was evolved.
